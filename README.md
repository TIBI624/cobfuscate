# CObfuscate

Python code obfuscator with C backend support.

## Installation

```bash
pip install cobfuscate
```

## Usage

```bash
# Obfuscate single file
cobfuscate input.py output.py

# Obfuscate directory
cobfuscate ./src ./obfuscated
```

## Requirements

- Python 3.7+
- clang (for C extension)
